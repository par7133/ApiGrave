<h1>ApiGrave</h1>
Your api, our  grave. Let's wrap your php api online with ease..  <br><br>

Hello and welcome to ApiGrave!<br>
	   
ApiGrave is a light, simple, software on premise for "burying" your api online. Just drop your .inc files in the Api folder and you are all set: your api is now live. However check the config file and fill there with your values, there is also space for Memcached.<br>
